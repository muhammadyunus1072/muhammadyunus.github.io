Tentang Saya

Halo, nama saya Muhammad Yunus. Saya tinggal di Pasuruan, Jawa Timur. Menjadi alumni SMKN 1 Pasuruan di jurusan Rekayasa Perangkat Lunak (RPL) pada tahun 2020. Saya Lahir pada 12 Oktober 2001, usia saya saat ini 20 tahun. Alasan saya memilih jurusan RPL karena pada waktu SMP saya tertinggal tentang Teknologi dan Media Sosial, Oleh karena itu saya ingin belajar dan mengetahui lebih banyak tentang Teknologi.

Saat di SMK saya pernah mewakili Sekolah dan Kota Pasuruan untuk mengikuti Lomba Kompetensi Siswa (LKS) Jawa Timur di Jember. Setelah mengikuti LKS tersebut saya senang karena pengetahuan saya tentang Teknologi Pengembangan Web bertambah. Sekarang saya terus belajar dan berharap mendapatkan kesempatan untuk terjun langsung bekerja di dunia Pengembangan Web. Demikian cerita singkat tentang saya, Terima kasih.
